API Documentation

API Documentation - Tên Dự Án

1. Tổng Quan

API này cung cấp các chức năng để quản lý [mô tả API].

BASE_URL: http://localhost:5000/api/v1

2. Authentication (Xác thực)

Tất cả các API yêu cầu xác thực qua JWT Token (trừ đăng nhập, đăng ký).

Authorization: Bearer <your_token_here>

3. API Endpoints

Headers:

Authorization: Bearer <your_token_here>

Request Body:

{}

Response (200):

{
    "success": true,
    "message": "Account deleted successfully"
}

Response (500 - Error):

{
    "success": false,
    "message": "Error message"
}

Notes:

  • Chỉ những "recruiter" đã đăng nhập mới có thể xóa tài khoản của họ.
  • API yêu cầu xác thực thông qua JWT Token trong phần header "Authorization".

4. Lỗi Thường Gặp

Mã lỗi Mô tả lỗi
400 Request không hợp lệ (Bad Request)
401 Chưa xác thực (Unauthorized)
403 Không có quyền truy cập (Forbidden)
404 Không tìm thấy tài nguyên (Not Found)
500 Lỗi máy chủ (Internal Server Error)
